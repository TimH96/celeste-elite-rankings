Celeste Elite Rankings

About

This website ranks Celeste speedrunners by an accumulated score from all their Individual Level times, largely inspired by the-elite.net. For the data source, it uses speedrun.com and its REST API.

It is pretty basic right now, by design. With enough interest, I can see myself investing more time to make it prettier and add features. And of course it is open source, you are free to submit your own changes via a PR. Here is the repository.

Ranking

The ranking algorithm works like this for each IL board:
1st gets 100 points, 2nd gets 97 points, 3rd gets 95 points - and from there its linear, meaning 4th gets 94 points and so on. All individual scores are summed up for each player, resulting in their final point count.

Note: Farewell is considered an A-Side in this context. Also, Collectible runs use the Full Clear subcategory.
